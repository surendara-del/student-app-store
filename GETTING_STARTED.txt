🆘 TROUBLESHOOTING

Problem: Port 5000 already in use
Solution: Edit app.py and change port number

Problem: Dependencies not installing
Solution: Update pip first: pip install --upgrade pip

Problem: File upload not working
Solution: Ensure uploads folder exists and has write permissions

Problem: Database errors
Solution: Delete app_store.db and restart (it will recreate)

Problem: JavaScript not working
Solution: Clear browser cache or use incognito mode
